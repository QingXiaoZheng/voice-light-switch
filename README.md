# 语音控制机械臂关灯系统

## 目标
使用小米音箱 Pro、ESP32 开发板和舵机，实现语音控制关灯的功能，适合动手能力低的用户。

---

## 方案概览
1. **语音命令输入**：小米音箱作为语音助手，用户通过语音控制触发操作。
2. **控制器**：ESP32 通过 Wi-Fi 接收音箱指令，并控制舵机动作。
3. **舵机动作**：舵机推动灯开关实现关灯动作。
4. **电源供给**：使用小型移动电源供电，无需接入墙插。

---

## 硬件清单
1. **ESP32 开发板（1 个）**
   - 型号：ESP32-WROOM（支持 Wi-Fi 和蓝牙）。
   - **链接到电源**：使用 5V USB 数据线连接到移动电源。

2. **舵机（1 个）**
   - 推荐：**MG90S**（金属齿轮，力矩更高）或 **SG90**（塑料齿轮，成本低）。
   - **作用**：掌控机械臂，按压开关。

3. **舵机安装支架（1 套）**
   - 简单机械臂组件，用于固定舵机并使其推到开关。

4. **移动电源（1 个）**
   - 推荐容量：5000mAh。
   - 用于为 ESP32 和舵机提供电源。

5. **固定材料**
   - 双面胶、热熔胶枪（固定舵机和支架）。

## 执行步骤

### 步骤 1：硬件安装
1. 将舵机固定在灯开关附近：
   - 调整舵机臂的长度和角度，使其能够触碰并推动灯的开关。
   - 使用支架（自带螺丝或双面胶）固定舵机。

2. 用杜邦线连接舵机与 ESP32：
   - 舵机引脚对应：
     - 橙线（信号）：GPIO 引脚（如 D5）。
     - 红线（电源）：ESP32 的 5V 引脚。
     - 棕线（地线）：ESP32 的 GND 引脚。

3. 确保 ESP32 与舵机位置稳定，并连接移动电源。

### 步骤 2：集成小米音箱
1. 打开米家 APP，创建智能场景：
   - 触发条件：语音指令“小爱同学，关灯”。
   - 动作：HTTP 请求，地址为 `http://<ESP32-IP地址>/turn_off`。

2. 测试语音命令：
   - 确保指令触发后，舵机执行动作并成功按下开关。
